# Lightweight Go Game ⚫⚪

A lightweight Go game built from scratch with a focus on clean architecture, strategic gameplay, performance, and core board game fundamentals.

## 🎯 Project Goal

The goal of this project is to create a simple and efficient Go implementation while understanding complex board systems, territory control, and strategic decision-based gameplay.

## ✨ Features

* Go board system
* Stone placement mechanics
* Turn management
* Liberty detection
* Capture system
* Territory calculation
* Legal move validation
* Game state tracking
* Move history

## 🏗️ Architecture

The project focuses on separating important game components:

* Game Loop
* Board System
* Stone System
* Rule Engine
* Liberty Detection System
* Capture System
* Territory System
* Game State Manager
* Input System
* Rendering System
